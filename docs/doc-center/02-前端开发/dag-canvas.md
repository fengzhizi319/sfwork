# 页面需求：DAG 画布（项目空间 /dag）

## 页面目标
提供一个可视化、交互式的隐私计算训练流编排环境，让建模工程师通过拖拽组件、配置参数、连接节点完成算法流程，并支持运行、调试与结果查看。

## 用户故事
- 作为建模工程师，我希望从左侧组件库拖拽组件到画布。
- 作为建模工程师，我希望连接组件的输入输出端口，系统帮我校验类型是否匹配。
- 作为建模工程师，我希望选中组件后在右侧配置参数。
- 作为建模工程师，我希望保存、运行、停止训练流，并查看日志与结果。

## 左侧面板

### 训练流树
- 展示当前项目下所有训练流（Graph）。
- 支持创建、重命名、复制、删除、导出。
- 当前训练流高亮。
- P2P 模式下，非本方创建的训练流仅可查看。
- 归档项目不可编辑训练流。

### 组件库
- 按分类目录树展示 SecretFlow 组件（如 数据读写、特征工程、统计、联邦学习、模型评估等）。
- 搜索组件。
- 拖拽组件到画布。
- 悬停显示组件文档说明（支持 i18n）。

### 数据集
- 展示当前项目已授权的数据表。
- 拖拽数据表节点到画布，自动作为读数据节点。

## 画布区域

### 节点
- 节点外观：图标 + 组件名 + 状态色环。
- 节点状态色：
  - 灰色：未运行
  - 蓝色：运行中
  - 绿色：成功
  - 红色：失败
  - 黄色：停止
- 节点支持拖拽、缩放、多选、框选。

### 连线
- 从输出端口拖拽到输入端口建立连接。
- 连接校验：数据类型匹配、禁止自环、禁止重复连接同一端口。
- 选中连线可删除。

### 顶部工具栏

| 按钮 | 功能 |
|---|---|
| 保存 | 持久化当前画布 |
| 运行 | 按选中节点拓扑运行 |
| 停止 | 停止当前运行中的作业 |
| 放大/缩小 | 画布缩放 |
| 适应屏幕 | 自动缩放到全部节点可见 |
| 撤销/重做 | 操作历史回退 |
| 高级配置 | 打开全局运行配置 |
| 周期任务 | 打开周期任务入口 |
| 模型提交 | 打开模型提交抽屉 |

## 右侧抽屉

### 组件配置抽屉
- 组件类型、版本、ID。
- 参数表单：根据组件元数据动态渲染输入框、下拉框、表格等。
- 自定义渲染项：如分箱修改、线性模型参数修改。
- 支持模板快速配置。

### 记录与结果抽屉
- 展示当前训练流的执行记录列表。
- 点击记录可查看 DAG 快照与每个节点的输出。

### 模型提交抽屉
- 选择要提交的模型组件。
- 显示提交进度。
- 退出时提示会中断提交进程。

### 日志抽屉
- 查看任务/节点日志。
- 支持 SLS/ELK 等外部日志跳转。

## 操作与反馈

| 操作 | 反馈 |
|---|---|
| 拖拽组件到画布 | 生成默认节点，自动分配 ID 与序号 |
| 连接节点 | 类型不匹配时连线失败并 Toast 提示 |
| 保存 | 成功/失败 Toast，失败时保留编辑状态 |
| 运行 | 弹出确认，开始轮询任务状态 |
| 运行失败 | 失败节点红色高亮，抽屉展示错误日志 |
| 停止 | 停止后所有运行中节点状态变为 STOPPED |

## 业务规则
- 画布编辑权限由 `ProjectEditService` 统一控制，受平台类型、项目归档状态、训练流创建方影响。
- 只有已授权到项目的数据表才会出现在数据集面板。
- 运行前会校验节点与路由健康。
- 选中节点运行只运行该节点及其上游依赖；未选中则运行整个画布。
- 内置组件（如读数据表）在 JobChain 中直接标记成功，不提交到 Kuscia。
