# 前端模板与参数 UI 设计

## 1. 目标

在 SecretPad 前端新增：

1. 「数据分类分级」训练流模板。
2. 模板在新建项目/新建训练流时可见。
3. 拖拽 `privacy/data_classification` 组件后，参数配置界面可正常显示自动模式/高级模式切换。

## 2. 训练流模板

### 2.1 文件位置

`secretpad/frontend-src/apps/platform/src/modules/pipeline/templates/pipeline-template-data-classification.ts`

### 2.2 模板内容

模板生成一个两节点 DAG：

```text
read_data/datatable  →  privacy/data_classification
```

- `read_data/datatable`：用户选择样本表。
- `privacy/data_classification`：自动模式下默认参数。

### 2.3 参数默认值

自动模式：

```ts
attrPaths: ['mode', 'mode/auto/default_level'],
attrs: [
  { s: 'auto', is_na: false },
  { s: 'L3', is_na: false },
],
```

## 3. 注册模板

需要修改以下文件：

1. `apps/platform/src/modules/pipeline/pipeline-protocol.tsx`
   - 枚举新增 `DATA_CLASSIFICATION = 'data-classification'`。
   - `TemplateIcon` 增加图标。
2. `apps/platform/src/modules/pipeline/index.ts`
   - `PipelineTemplatesRegistry.register(getModel(TemplateDataClassification))`。
3. `apps/platform/src/modules/pipeline/pipeline-creation-view.tsx`
   - 在模板列表渲染中加入新类型（通常与现有模板一起遍历，无需额外代码）。
4. `apps/platform/src/modules/pipeline/p2p-pipeline-creation-view.tsx`
   - 调整模板排序数组，把新模板加入合适位置。

## 4. 参数 UI

### 4.1 自动渲染

`privacy/data_classification` 组件的参数使用标准 SecretFlow 组件定义：

- `mode`: `AT_UNION_GROUP`，选项 `auto` / `advanced`。
- `auto/default_level`: `AT_STRING`，下拉选择 L1~L5。
- `advanced/enable_rule_engine`: `AT_BOOL`
- `advanced/enable_small_ner`: `AT_BOOL`
- `advanced/enable_llm`: `AT_BOOL`
- `advanced/icd10_l4_intervals_json`: `AT_STRING`
- `advanced/manual_override_json`: `AT_STRING`

前端 `config-form-view.tsx` 会按组件定义自动渲染为表单。`AT_UNION_GROUP` 会渲染为一个下拉选择框，选择不同模式后自动显示/隐藏对应分支参数。

### 4.2 自定义面板样式（可选）

若默认渲染不满足体验，可新增面板样式注册：

`apps/platform/src/modules/component-config/component-panel-style-registry/mpc/data-classification.ts`

配置项：

- `mode` 排序第一。
- advanced 分支参数标记 `isAdvancedConfig: true`（如果希望默认折叠）。

然后在 `component-panel-style-registry/index.ts` 中注册。

本 V1 实现不强制要求自定义面板样式，依赖默认渲染即可。

## 5. 图标

复用现有隐私计算图标 `SafetyOutlined`（与差分隐私一致），或改用 `FileSearchOutlined`。

## 6. 验证方式

1. 启动前端 dev server。
2. 新建项目时看到「数据分类分级」模板。
3. 选择模板后 DAG 画布出现两个节点。
4. 点击 `privacy/data_classification` 节点，右侧配置抽屉出现「模式」下拉框，可切换自动/高级。
