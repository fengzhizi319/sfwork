## 文档来源

本文档中心整合自以下目录：

- `sfwork/docs/` —— 项目级文档与白皮书
- `sfwork/secretpad/docs/` —— SecretPad 后端文档
- `sfwork/secretpad/frontend-src/` —— SecretPad 前端源码内文档
- `sfwork/secretpad-frontend/` —— SecretPad 前端独立仓库文档
- `sfwork/kuscia/docs/` —— Kuscia 官方文档
- `sfwork/privacy-local-agent/docs/` —— 本地隐私 Agent 文档

> 注：复制时部分文件名加了来源前缀（如 `secretpad-`、`kuscia-`、`frontend-`）以避免重名并便于识别来源。

## 使用建议

1. **先读总览**：从 `00-项目总览/` 的白皮书和 PPT 建立整体认知。
2. **再按模块深入**：根据角色进入 `02-前端开发/`、`03-后端开发/` 或 `04-Kuscia/`。
3. **交叉参考**：架构设计、部署运维、问题排查通常需要跨模块阅读。
4. **以原始文档为准**：本中心为归档副本，最新内容请回源仓库确认。
