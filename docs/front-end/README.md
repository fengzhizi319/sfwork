## 核心产品目标

SecretPad 前端是隐私计算平台的**可视化控制中心**，让数据科学家、算法工程师、平台管理员和机构协作者在不需要理解底层 MPC/TEE/Kuscia 细节的情况下，完成：

1. **节点与机构管理**：注册节点、建立机构间路由、查看节点健康。
2. **数据资产管理**：注册数据源、创建数据表、授权到项目、加密上传到 TEE。
3. **隐私计算项目**：创建项目、选择参与方、编排训练流（DAG）、运行与监控任务。
4. **模型与结果**：模型打包、发布在线服务、下载结果、查看报告。
5. **协作审批**：项目邀约、数据授权、节点合作、TEE 下载等消息与投票处理。
