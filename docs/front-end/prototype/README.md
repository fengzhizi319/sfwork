# SecretPad 高保真原型

> 本目录是基于 PRD 实现的高保真静态原型，可直接在浏览器中打开预览，也可作为素材导入 Figma / 墨刀继续精化。

## 在线预览

直接用浏览器打开 `index.html` 即可。

```bash
# macOS
open docs/front-end/prototype/index.html

# Linux
xdg-open docs/front-end/prototype/index.html
```

## 原型内容

左侧导航可切换以下页面：

1. **Dashboard**：统计卡片、最近动态、系统健康。
2. **项目管理**：项目卡片列表、创建项目抽屉。
3. **节点管理**：节点表格、新增节点抽屉。
4. **数据源**：数据源表格、新增数据源抽屉。
5. **数据表**：数据表表格、新增数据表抽屉。
6. **DAG 画布**：顶部二级菜单、工具栏、节点与连线示意。
7. **消息中心**：Tab 切换、消息列表、消息详情抽屉。
8. **模型管理**：模型列表与服务操作。
9. **结果管理**：结果列表与下载。

## 导入设计工具

### 导入 Figma
1. 在浏览器中打开 `index.html`，使用浏览器截图工具截取各页面。
2. 在 Figma 中新建 Frame，将截图作为底图放置。
3. 使用 Figma 组件库（Ant Design / Arco Design）覆盖还原为可编辑矢量元素。
4. 抽屉状态可单独截图或新建 Frame 设计。

### 导入墨刀
1. 在墨刀中新建“自由画布”或“Web”项目。
2. 将截图拖入画布作为参考底图。
3. 使用墨刀内置组件（按钮、表格、卡片、表单）按参考图还原。
4. 为左侧导航、表格行、按钮添加页面跳转/弹窗交互。

## 文件说明

| 文件 | 说明 |
|---|---|
| `index.html` | 单页高保真原型，含全部页面与抽屉 |

| `app.js` | 简单的页面切换与抽屉交互 |
| `README.md` | 本说明 |

## 设计规范摘要

- 主色：`#1677ff`（Ant Design 蓝）
- 成功：`#52c41a`，警告：`#faad14`，错误：`#f5222d`
- 卡片圆角：8px，按钮圆角：6px
- 侧边栏宽度：220px，顶部 Header 高度：56px
- 表格、表单、标签等遵循 PRD `00-global-spec.md`
