## 5. DAG 画布（/dag）

```textn+------------------------------------------------------------------+
|  项目：反欺诈联邦建模    项目数据 | 模型训练 | 模型管理 | 周期任务  |
+------+------------------------------------------+------------------+
| 左侧  |                                          | 右侧抽屉（组件配置）|
| 面板  |           DAG 画布                        |                  |
|       |                                          | 组件：样本读取     |
| 训练流 |    +---------+      +---------+         | - 数据表: user   |
| 组件库 |    | 读数据   |----->| 特征工程 |         | - 列选择...      |
| 数据集 |    +---------+      +---------+         |                  |
|       |          |              |               | [保存] [取消]    |
|       |          v              v               |                  |
|       |    +---------+      +---------+         |                  |
|       |    | 联邦训练 |----->| 模型评估 |         |                  |
|       |    +---------+      +---------+         |                  |
|       |                                          |                  |
|       |  [保存] [运行] [停止] [撤销] [适应] [+/-] |                  |
+------+------------------------------------------+------------------+
```
